Glitchified VilniusTech
=======================

Šis projektas yra meninė interpretacija, pagrįsta VilniusTech svetainės HTML struktūra.
Į šį statinį puslapį yra įterptas `content.js`, kuris įgyvendina vizualinius, tekstinius ir performatyvius efektus.

✅ Paleidimui:
1. Įkelk šį aplanką į GitHub repozitoriją.
2. Įjunk GitHub Pages (pvz. /docs arba main branch).
3. Atverk viešą URL, kad pamatytum glitch meną.

- JS failas: content.js
- Video sąrašas: brainrot_videos.js
- Puslapio failas: index.html
